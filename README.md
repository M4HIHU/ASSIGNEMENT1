#  Online Library Management System (C++)

## Course
BSD 122: Object-Oriented Programming 1

## Description
This project is an Online Library Management System developed using Object-Oriented Programming (OOP) principles in C++.  
It allows management of books and users, including borrowing and returning of books.

## Features
- Add and remove books
- Register users
- Search books by title
- Borrow and return books
- Handles invalid operations gracefully

## OOP Concepts Used
- Encapsulation
- Abstraction
- Modularity
